# Learning

## Purpose
프로젝트의 아키텍처와 패턴에 대한 학습 자료를 담은 디렉토리입니다. DataSource 패턴, NestJS DI, 도메인 기반 설계 등 핵심 개념을 단계별로 설명합니다.

## Key Files
- `01-data-access-patterns.md` - 데이터 접근 패턴 기초
- `02-nestjs-di-concepts.md` - NestJS 의존성 주입 개념
- `03-codebase-interface-analysis.md` - 코드베이스 인터페이스 분석
- `04-codebase-factory-analysis.md` - 팩토리 패턴 분석
- `05-codebase-module-service.md` - 모듈/서비스 구조 분석
- `06-extension-guide.md` - 확장 가이드 (새 DataSource 추가 등)
- `07-domain-based-aggregation.md` - 도메인 기반 집계 패턴

## For AI Agents
- 새로운 기능 구현 전 관련 학습 자료 참고
- 번호 순서대로 읽으면 점진적 학습 가능
- 코드베이스 분석 결과와 실제 코드 비교 학습에 유용
