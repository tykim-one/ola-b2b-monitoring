## 빠른 시작

### 로컬 개발 환경 실행
```bash
# 의존성 설치
pnpm install

# 전체 스택 실행 (백엔드 + 프론트엔드)
pnpm dev:all
```

### 접속 URL
- **프론트엔드**: http://localhost:3001
- **백엔드 API**: http://localhost:3000
- **기본 관리자 계정**: admin@ola.com / admin123
