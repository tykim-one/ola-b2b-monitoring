# public

## Purpose
정적 자산 디렉토리입니다. Next.js의 public 폴더로 빌드 시 루트 경로에 직접 서빙됩니다.

## Key Files
- `file.svg` - 파일 아이콘
- `window.svg` - 윈도우 아이콘
- `globe.svg` - 글로브 아이콘
- `vercel.svg` - Vercel 로고
- `next.svg` - Next.js 로고

## For AI Agents
- 이미지/아이콘 추가 시 이 디렉토리에 배치
- URL 접근: `/파일명` (예: `/globe.svg`)
