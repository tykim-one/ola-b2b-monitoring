# Generated

## Purpose
자동 생성된 코드를 포함하는 디렉토리입니다. 주로 Prisma 클라이언트 등 ORM/코드 생성 도구의 출력물이 위치합니다.

## Subdirectories
- `prisma/` - Prisma 생성 클라이언트 코드

## For AI Agents
- **절대 수동 편집 금지**: 이 디렉토리의 파일들은 자동 생성됨
- 스키마 변경 시 `pnpm prisma:generate` 재실행 필요
- 타입 참조만 가능, 직접 수정 불가

## Dependencies
- `prisma/schema.prisma` 스키마 파일
- Prisma CLI (`pnpm prisma:generate`)
